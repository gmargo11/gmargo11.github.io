Learning to See Physical Properties with Active Sensing Motor Policies

Gabriel B. Margolis     Xiang Fu     Yandong Ji     Pulkit Agrawal

Conference on Robot Learning (CoRL) 2023




Abstract

To plan efficient robot locomotion, we must utilize the information about a terrain's physics that can be inferred from its color, texture, and semantics. To this end, we train a visual perception module that predicts terrain properties using labels from a small amount of real-world proprioceptive locomotion. To ensure label precision, we introduce Active Sensing Motor Policies. These policies are trained to prefer motor skills that facilitate accurately estimating the surrounding environment's physical attributes. The estimated labels supervise a vision model that infers physical properties directly from color images and can be reused for different tasks. Leveraging a pretrained vision backbone, we demonstrate robust generalization in image space, enabling path planning from overhead imagery despite using only ground camera images for training. Our method supports efficient robot navigation across diverse terrains and conditions.
